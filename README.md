# BinarySwap

Write a function that returns the minimum number of swaps to convert the first binary string into the second.
Examples:
minSwaps("1100", "1001") ➞ 1

minSwaps("110011", "010111") ➞ 1

minSwaps("10011001", "01100110") ➞ 4
